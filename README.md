# OnlineBookStore

Online Bookstore is a Java web application built with Maven, JSP, Servlets, and MySQL.

It provides a simple online book shopping experience with:
- User registration and login
- Admin management capabilities
- Book browsing by category
- Cart handling
- Order placement

The project is designed as a dynamic web application that runs on a servlet container such as Apache Tomcat and uses a MySQL database to store user and book data.
